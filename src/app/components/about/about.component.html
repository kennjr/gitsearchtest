About

A web app where you can search for a Github user by their username or a Github repository by its repo name.


No Authentication is required to use the application.


The application gets all of its data from the Github API. It's greate for quick Github searches and gives you the most important information on a repo or user


How to use:

Enter the keyword into the search field then press enter. Click on any of the user results for more info or View in Github for repo results.


No user data is being collected by us(the same can't be said for Github).
